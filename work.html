Chat-App

Chat App is a native application for both Android and iOS mobile devices, and provides users with a chat interface and options to share images and their location. The application was developed using Expo and written in React Native.

The Gifted Chat Library is used to create chat interface with customized message bubbles. Users can choose a theme color on the start screen before joining the chat, images as well as their current location in a map view can be sent.

Google firebase is used to authenticate users anonymously and data is stored using both the cloud and local storage.
